Documentação e explicação

Este website é um pequeníssimo projeto criado como trabalho da disciplina Fundamentos WEB do curso Sistemas de Informação. O projeto tem como objetivo demonstrar o uso de uma API estática do IBGE. Ao preencher o formulário, é possível escolher um estado e uma cidade do estado selecionado.

Esta api é uma REST API, isso significa que é estática, não necessitando de algum método ou chave API. Basta referenciar o link da API e filtrar os dados selecionados.
Aqui está um exemplo de quatro links ou referências:

  • Estados: Clique aqui para abrir o caminho da API que contém todos os estados do Brasil.
    Este caminho foi utilizado para listar os estados no campo "Estados".
    Documentação: Clique aqui.
  • Municípios: Clique aqui para abrir o caminho da API que contém todas as cidades do Brasil.
    Este caminho não foi utilizado, pois não há necessidade de listar todos os municípios do Brasil. Neste projeto, precisamos apenas dos municípios do estado selecionado
    Documentação: Clique aqui.
  • Municípios por estado: Clique aqui para abrir o caminho da API que contém todas as cidades do estado de Santa Catarina no Brasil.
    Este caminho foi utilizado para selecionar apenas os municípios de um estado específico, para isso, devemos utilizar uma variável e colocar na URL. Veja linha 13 do arquivo index.js para mais detalhes
    Documentação: Clique aqui
